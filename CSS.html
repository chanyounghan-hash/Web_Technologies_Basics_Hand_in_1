Media Queries

Breakpoints allow layout to adapt to different screen sizes. This project uses two breakpoints: 768px (tablet) and 480px (mobile). 

/* Tablet */
@media (max-width: 768px) {
  .nav {
    flex-wrap: wrap;
  }
}

/* Mobile */
@media (max-width: 480px) {
  .nav {
    flex-direction: column;
    align-items: flex-start;
  }
}

Basic HTML + CSS Template

This is a minimal pattern for a static site with an external stylesheet.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My Website</title>
    <link rel="stylesheet" href="styles.css" />
  </head>
  <body>
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
  </body>
</html>
