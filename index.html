Basic HTML Rules

Rule Explanation
Proper nestingTags must close in reverse order.
Use semantic elementsPrefer meaningful tags like <nav>, <main>, <section>.
Always close tagsUse <p></p> and close container elements properly.
Use alt for imagesRequired for accessibility and for cases where images do not load.
Lowercase syntaxUse lowercase tag and attribute names for consistency.
IndentationUse consistent indentation (2 or 4 spaces) for readability.
